This project conduced a meta-analysis of neuroimaging features associated with MeSH term "psychological resilience" (D055500).

Our systematic of psychological resilience was curated in Zotero and can be found at this address: https://www.zotero.org/groups/4721296/resilience_systematic_review/library

Custom python code was written to traverse the cureated library and extra regions associated with psychological resilience for subsequent meta-analysis in GingerALE.  Python code can be found at:  https://github.com/gandrewjames/resilience_metaanalysis

Meta-analytic findings can be found at OpenNeuro.org at:  https://openneuro.org/datasets/ds006138/versions.14-APR-2025
